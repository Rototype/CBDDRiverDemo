MarkCheckBehaviour

Define the identification marks behaviour according to its value (not applicable to CBD500/800):

0 - Use machine configuration. (default)

1 - Disable mark check (OMN).

2 - Enable covers only - marks are unique per sheet (OMY, OIY, OUY).

3 - Enable covers and cheques - marks are unique per sheet (OMY, OIN, OUY).

4 - Enable covers only - front cover is unique (OMY, OIY, OUN, OLY).

5 - Enable covers and cheques - front cover is unique (OMY, OIN, OUN, OLY).

6 - Enable covers only - back cover is unique (OMY, OIY, OUN, OLN).

7 - Enable covers and cheques - back cover is unique (OMY, OIN, OUN, OLN).

MicrCheckBehaviour

Define the codeline check behaviour according to its value (not applicable to CBD500/800):

0 - Use machine configuration. (default)

1 - Disable codeline check (ORN).

2 - Enable codeline check (ORY).

NoBookletExpose Activate the back-office option. This option changes the behaviour of the PrintBooklet method. The booklet is never exposed and the completion is given when the booklet is safe in the retract bin. Default is false. Not applicable to CBD500/800.
NoRibbon Activate the book issuing without binding tape application. Default is false. Not applicable to CBD500/800.
PersonalisedBitmaps List of the Volatile Bitmaps composing the booklet.
