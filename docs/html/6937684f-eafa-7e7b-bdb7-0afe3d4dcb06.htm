ChangePin Open the paper feeder cover lock. Require optional hardware.
Clear Try to remove an error condition that halted the decoding process, if the method returns false, completion and result will be reported asynchronously by the CompleteEvent event. The successfully completion will be given accordling to the termination of clear command.
Close Close the connection between the machine and the related serial port. Completion will be reported asynchronously by the DisconnectedEvent event.
EjectBooklet Eject the exposed booklet, if the method returns false, completion and result will be reported asynchronously by the CompleteEvent event. The successfully completion will be given when the booklet is safely ejected. Not applicable to CBD500/800.

ForcePaperSize Force a sheet size different fron the one specified in the template file. A parameter set to zero restores the size from the template. This command is not intended for normal use and can cause malfunctioning.

GetRetries Get the Maximum number of retries for some jamming errors. Not applicable to CBD500/800. The retries are in an array of Uint16. Index = 0 -> Double Feed (flag 'D'); Index = 1 -> Mark Mismatch (flag 'M'); Index = 2 -> Micr Error (flag 'R'); Index = 3 -> Preprinted Mismatch (flag 'N'); Index = 4 -> Sheets Near Empty in Feeder (flag 'P');

LoadBookletLayout(String) Load a layout booklet definition. The layout must be sent before printing the booklet. If the layout do not change it can be sent just once. Note that especially if there are bitmaps defined in the layout this command can take a long time to complete. This version applies to CBD1000 only, CBD 2000 requires the version with Booklet layout (for personalised bitmaps)
LoadBookletLayout(String, CbdDriverBooklet) Load a layout booklet definition. The layout must be sent before printing the booklet. If the layout do not change it can be sent just once. Note that especially if there are bitmaps defined in the layout this command can take a long time to complete. This version applies to CBD2000 and sends also personalised bitmaps (CBD 2000 operation is triggered by layout file extension (RBD)). This command has not a completion event associated.

Open Extablish a connection to the machine opening a new serial port connection using the specified port name according to the machine model defined by the CurrentModel property. If the method returns false, the completion will be reported asynchronously by the ConnectedEvent event.
OpenCover Open the paper feeder cover lock. Require optional hardware.
PrintBooklet Start the printing of a booklet. The layout booklet definition must be already loaded and the booklet data must be filled accordingly. If the method returns false, completion and result will be reported asynchronously by the CompleteEvent event. The successfully completion will be given accordling to the parameter when the booklet is exposed and ready to be taken, or when the booklet is safe in the retract box.
Recover Try to recover an error condition (Jammings, failures or Errors), if the method returns false, completion and result will be reported asynchronously by the CompleteEvent event. The successfully completion will be given accordling to the termination of recovered command.
RetractBooklet Retract the exposed booklet, if the method returns false, completion and result will be reported asynchronously by the CompleteEvent event. The successfully completion will be given when the booklet is safe in the retract box. Not applicable to CBD500.
SetRetries Set the Maximum number of retries for some jamming errors. The retries are in an array of Uint16. The allowed values ranging 0 to 25. Index = 0 -> Double Feed (flag 'D'); Index = 1 -> Mark Mismatch (flag 'M'); Index = 2 -> Micr Error (flag 'R'); Index = 3 -> Preprinted Mismatch (flag 'N'); Index = 4 -> Sheets Near Empty in Feeder (flag 'P');
