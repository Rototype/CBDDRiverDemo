CbdDriverLoadBookletLayout(String, CbdDriverBooklet) Method

Load a layout booklet definition. The layout must be sent before printing the booklet. If the layout do not change it can be sent just once. Note that especially if there are bitmaps defined in the layout this command can take a long time to complete. This version applies to CBD2000 and sends also personalised bitmaps (CBD 2000 operation is triggered by layout file extension (RBD)). This command has not a completion event associated.
